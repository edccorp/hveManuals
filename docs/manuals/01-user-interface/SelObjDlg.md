# Select Object Dialog Box

This dialog is not displayed on its own; it is a child panel embedded into the standard Windows file Open/Save dialog when HVE opens or saves objects and cases. It adds HVE-specific information alongside the standard file controls.

The panel contains the following controls:

## Case Title

An editable field containing the title stored with the case. The title is saved in the case file and is displayed in reports.

## Object Text

A label describing the object being opened or saved (its content depends on the current mode — human, vehicle, environment, event or playback).

## File List

A sorted list box displaying the existing files of the appropriate type in the current folder.

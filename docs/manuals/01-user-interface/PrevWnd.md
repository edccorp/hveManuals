# Report Window Dialog Box

This dialog (formerly titled *Set Preview Window*) is used when adding a playback window that displays a report for an event. It is used to select the event and the type of report to display.

This dialog contains the following options:

## Active Events

Displays the list of events currently existing in the case. Select the event whose report you wish to display.

## Status

A read-only field indicating whether the selected event has been run. Reports are only available for events that have been executed.

## Name

Contains the name for the report window. It defaults to the event currently selected in the Active Events list and may be edited.

## Report Type

A combo box containing all the available reports for the selected event. The user can select any report.

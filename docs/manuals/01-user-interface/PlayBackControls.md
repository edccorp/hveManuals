# Playback Controls (Toolbar)

The playback controls on the toolbar control execution of the current event during Event mode, and control replay during Playback mode. The following controls are available, in toolbar order:

## Reset

Initializes the physics program and places the simulation at its starting conditions.

## Rewind

Places the simulation at the starting position.

## Reverse

Plays the simulation backwards.

## Pause

Pauses the simulation.

## Play

Starts (or resumes) the simulation.

## Advance to End

Places the current simulation state at the end of the simulation.

## Max Sim Time

Displays the maximum simulation time, i.e., the time at which the simulation will terminate.

## Integration Time Step

Displays the time step used by the calculation method's numerical integration.

## Output Time Interval

Displays the time interval at which output results (and playback frames) are produced.

## Current Time

Displays the current simulation time.

## Frame No

Displays the number of the frame currently being displayed.

## Time Slider

A slider used to set the current simulation/playback time directly; drag it to move to any point in the event.

# Set Grid Dialog Box

Sets a user-definable distance interval for display and data entry in the environment. Grid markers are used as spatial cues to help determine distances while creating and editing 3-D models of humans, vehicles and environments.

This dialog box contains the following options:

## Turn Grid On

Check box that turns the grid display on or off.

## Grid Dimension (ft)

The distance between grid lines or markers in the 3-D editor viewers, in the current user units.

# Text Editor Dialog Box

The Text Object Tool allows the user to create and edit 3-D text. Text may be used to label signs, buildings and other objects. The Text Editor dialog displays information about the current text object.

## Coords (ft) — X, Y, Z

The coordinates of the text object.

## Angle (deg) — Roll, Pitch, Yaw

The orientation of the text object.

## Global / Local

A combo box selecting whether the entered coordinates and angles are interpreted in the global (environment) coordinate system or the text object's local coordinate system.

## Text

Allows the user to enter or edit the desired text string. The field is multi-line, so labels may contain several lines of text.

## Height (ft)

Allows editing of the text height.

## Apply

Applies the current settings to the selected text object.

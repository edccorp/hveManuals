# Overlays Dialog Box

Choosing Overlays from the View menu displays the Overlays dialog. The Overlays dialog allows the user to display or remove objects from the scene.

## Active Overlay List

This multiple-selection list box displays all the overlay names of the objects in the viewer. By default, all overlays are selected, and therefore all objects are displayed in the viewer. Click on an overlay name to remove all objects using that name; click it again to restore them.

## Scanned Image Overlay

The Scanned Image check box is used to display or remove a scanned image from the environment. A scanned image may be used for the entire environment, or just the sky. The check box is greyed out (disabled) unless the environment actually contains a scanned image (backdrop).

# Sky Attributes Dialog Box

Sky attributes determine the appearance of the background when a 3-D geometry file is displayed.

The following attributes are available:

## Ambient Light Intensity

A slider and edit field that determine the ambient lighting level in the scene. This attribute may be used for visualizing various levels of daylight, dusk/dawn and darkness.

## Set Sky Color

Displays the Color dialog box. Use the controls in the dialog box to choose the desired color for the sky.

## Fog Type

A user-selectable type of visual occlusion:

- **None** — No visual occlusion.
- **Haze** — Selects the visual occlusion as haze.
- **Fog** — Selects the visual occlusion as fog.
- **Smoke** — Selects the visual occlusion as smoke.

## Visibility

A user-editable field defining the distance (from the camera) at which objects are no longer visible.

## Set Fog Color

Displays the Color dialog box, used to set the color of the fog/haze/smoke.

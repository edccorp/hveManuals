# Collision Pulse In-Use Factors Dialog Box

It may be useful to study the sensitivity of the occupant motion to changes in the magnitude of the collision pulse. The Pulse Factors dialog (title: *Collision Pulse In-use Factors: [vehicle name]*) allows the user to quickly assign multipliers to each degree of freedom.

Each factor is entered as a multiplier (%/100) using a slider or its associated edit field. The allowed range is 0.0 to 2.0; the default value is 1.0 (no scaling).

## Forward (%/100)

Multiplier applied to the forward component of the vehicle's linear acceleration.

## Lateral (%/100)

Multiplier applied to the lateral component of the vehicle's linear acceleration.

## Vertical (%/100)

Multiplier applied to the vertical component of the vehicle's linear acceleration.

## Roll (%/100)

Multiplier applied to the angular acceleration about the vehicle's roll axis.

## Pitch (%/100)

Multiplier applied to the angular acceleration about the vehicle's pitch axis.

## Yaw (%/100)

Multiplier applied to the angular acceleration about the vehicle's yaw axis.

## OK / Cancel

OK validates and assigns the factors; Cancel discards the changes. The factors are reset to 1.0 when **New** is pressed in the Collision Pulse dialog.

# Common Reports Dialog Box

This window is used for displaying the numeric (text) output reports produced in Playback mode. It is a resizable, scrollable report window: the report text may be scrolled vertically and horizontally using the scroll bars or the mouse wheel. The window has no buttons; it is closed with its system menu close box.

The Common Reports window is used to display the following report types:

- **Accident History** — table of positions and velocities for the event (see the Accident History topic)
- **Damage Data** — vehicle damage results (CDC, PDOF, crush profile, delta-V, energy)
- **Driver Data** — driver control input tables (steering, braking, throttle, gear)
- **Environment Data** — description of the environment used by the event
- **Event Data** — event setup data (positions, velocities, event options)
- **Human Data** — human parameter data for humans in the event
- **Injury Data** — computed occupant injury measures
- **Messages** — informational, warning and error messages produced by the physics program
- **Program Data** — physics program name, version, calculation options and execution information
- **References** — technical references for the physics program
- **Results** — general simulation/reconstruction results
- **Vehicle Data** — vehicle parameter data for vehicles in the event
- **Event Notes** — user-entered notes attached to the event

Graphic reports (Site Drawing, Damage Profiles, Momentum Diagrams) are displayed in the Common Scene Reports window instead, and the Variable Output and Trajectory Simulation reports have their own dedicated windows.

# Variable Output: Graph Dialog

The Variable Output Graph dialog is displayed by pressing **Graph...** in the Variable Output report window. It produces an XY plot of the currently selected output variables versus time.

## Graph

The plot area fills most of the dialog. Up to the first six selected variables are graphed as separate series, each drawn in a distinct color with a legend entry showing the object and variable name. Values are plotted against simulation time on the X axis, in the current user units.

If more than six variables are selected in the Variable Output table, only the first six are graphed.

## Print...

Prints the current graph.

## OK / Cancel

Both buttons close the graph dialog and return to the Variable Output report window.

# Trajectory Report

The Trajectory Simulation output window is a dynamic visualization of the human and vehicle motion, the same as during Event mode, beginning at the user-entered initial position and continuing until the end of the simulation. Like other output reports, Trajectory Simulations are displayed in a resizable Playback window selected from the Playback Window (report preview) dialog.

The window consists of a single 3-D render area showing the animated event; it has no buttons of its own.

The motion in the Trajectory Simulation window is controlled using the Playback Controller (play, stop, step and the time slider on the toolbar). When more than one Trajectory Simulation window is open, the playback controls act on the currently active window. Only one Trajectory Simulation window may be opened per event.

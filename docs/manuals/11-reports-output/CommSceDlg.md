# Common Scene Report Dialog

This resizable window is used for displaying graphic output reports in Playback mode. The body of the window is a 3-D render area showing the selected report:

- **Site Drawing** — a scaled drawing of the environment
- **Damage Profiles** — the vehicle damage profiles (displayed in this window for reconstruction events; simulation events use the dedicated Damage Profiles report window)
- **Momentum Diagram (Scene)** — the momentum analysis diagram drawn over the scene
- **Momentum Diagram (Damage)** — the momentum analysis diagram drawn with the damage data

## Next

Visible only for Momentum Diagram reports. Advances the display to the next phase of the momentum analysis. The diagram is built up in six phases; pressing Next after the last phase wraps around to the first.

## Previous

Visible only for Momentum Diagram reports. Steps the display back to the previous phase of the momentum analysis, wrapping from the first phase to the last.

# New Ellipsoid Dialog

This dialog is used to add a new contact ellipsoid to the currently selected human. It is displayed by the Add button in the human's Contact Ellipsoids dialog.

## Enter a name for the new Ellipsoid

User-editable name for the new ellipsoid. The name must be 30 characters or fewer; a longer name produces a "name too long" error message and the dialog remains open. The name must also be unique among the human's existing ellipsoids — a duplicate name is rejected after the dialog closes.

## OK

Accepts the name and creates the new ellipsoid, provided the name is valid and the human does not already have the maximum of 3 contact ellipsoids (an error message is displayed if the maximum is exceeded).

## Cancel

Closes the dialog without adding an ellipsoid.

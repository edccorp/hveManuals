# Joint Data Dialog Box

Joint parameters for the HVE Human Model are defined for each of the 14 joints. Open the dialog by clicking on a segment CG and choosing Joints from the segment popup menu. The dialog contains an option list containing the names of each joint attached to the selected segment (a segment may have up to 4 attached joints), and the joint i, j, k coordinates relative to the selected segment coordinate system. The dialog title bar shows the current human's name and the selected segment.

## Joint
Drop-down list of the joints attached to the selected segment. Select a joint to display and edit its properties.

## Coordinates i, j, k
The i, j and k coordinates of the selected joint, defined relative to the segment principal axis system. Edited coordinates are range-checked when OK is pressed, and the human's 3-D geometry in the editor is updated to reflect the new joint positions.

## Properties
Displays the [Joint Properties Dialog Box](HumJntPropDlg.md) for the selected joint.

> Note: The old help linked the Properties button to this topic itself; the Joint Properties dialog is documented separately.

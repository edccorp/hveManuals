# Human CG Dialog Box

Clicking on the CG of any segment of a human displays this popup dialog, which shows a list of options that can be performed for the selected segment. The dialog title bar shows the name of the selected segment (Pelvis, Abdomen, Chest, Neck, Head, Rt/Lt Upper Leg, Rt/Lt Lower Leg, Rt/Lt Foot, Rt/Lt Upper Arm, Rt/Lt Lower Arm).

This dialog contains the following items:

**Inertias...**
Displays the [Inertial Data Dialog Box](Human.md) for the selected segment.

**Color...**
Displays the Color Editor dialog box, with which the user can set the color of the selected segment.

**Ellipsoids...**
Displays the [Human Contact Ellipsoids Dialog Box](Human1.md) for the selected segment.

**Injury Tolerance...**
Displays the [Injury Tolerance Data Dialog Box](Human2.md).

**Joints...**
Displays the [Joint Data Dialog Box](Human3.md) for the joints attached to the selected segment.

**Cancel**
Closes the popup without making a selection.

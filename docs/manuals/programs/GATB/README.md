## How to Use This Manual

This manual is the user's manual for the GATB simulation model. The manual describes the model features and use. Input parameters and output reports are described at a high level. This manual is intended to be used in conjunction with the HVE Operations Manual. The HVE Operations Manual provides in-depth coverage of the human, vehicle and environment models, as well as the HVE Event Editor and Playback Editor. For specific model definitions, the user is referred to the HVE Operations Manual (see also the updated pages under [`docs/manuals/07-humans/`](../../07-humans/README.md)).

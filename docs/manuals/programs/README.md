# Physics Program Manuals

Updated Markdown editions of the per-model program manuals, verified against the current physics engines in `Physics/Source/`. Part of the [HVE Manual](../README.md).

## Manuals

- [EDCRASH](EDCRASH/README.md) — reconstruction of vehicle collisions
- [EDGEN](EDGEN/README.md) — generic (kinematic) path/position model
- [EDHIS](EDHIS/README.md) — occupant (human interior simulation) model
- [EDSMAC](EDSMAC/README.md) — 2-D collision & trajectory simulation
- [EDSMAC4](EDSMAC4/README.md) — enhanced SMAC collision simulation
- [EDSVS](EDSVS/README.md) — single-vehicle (2-DOF) dynamics
- [EDVDS](EDVDS/README.md) — vehicle dynamics (trucks/articulated)
- [EDVSM](EDVSM/README.md) — vehicle stability & rollover (HVOSM)
- [EDVTS](EDVTS/README.md) — tractor-semitrailer dynamics
- [SIMON](SIMON/README.md) — 3-D multi-body simulation with DyMESH
- [DyMESH](DyMESH/README.md) — 3-D dynamic-mesh collision & damage model (used by SIMON / EDSMAC4)
- [GATB](GATB/README.md) — articulated total-body human model
- [RDF](RDF/README.md) — import external time-position data as an event
- [DamageStudio](DamageStudio/README.md) — damage profile / crush visualization

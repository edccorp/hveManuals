# Steering System Dialog

The steering system parameters for the current vehicle are displayed and edited using the Steering System dialog. The dialog title identifies the current vehicle.

The Steering System parameters are:

## Steering Data For

Selects the axle whose steering data you want to view or modify (*Axle No. 1*, *Axle No. 2* or *Axle No. 3*, limited by the number of axles defined in the Vehicle Information dialog).

## Steerable Axle

Check box to allow or disallow steering at the selected axle. When unchecked, all of the parameter fields below are disabled for that axle, and the Spindle Axis button in the Suspension Data dialog is disabled for that axle as well.

## Steering Gear Ratio (deg/deg)

Ratio of the steering wheel angle to the road wheel steer angle.

## Steering Column Friction (in-lb)

Friction torque measured at the steering wheel.

## Steering Column Inertia (lb-sec²-in)

Rotational inertia of the steering wheel and column.

## Steering Column Stiffness (ft-lb/deg)

Torsional stiffness of the steering system between the steering wheel and the wheel on the driver's side.

## Steering Linkage Mass (lb-sec²/in)

Mass (weight/g) of all translating linkage components.

## Steering Linkage Stiffness (ft-lb/deg)

Torsional stiffness of the steering system between the driver's-side wheel and the opposite-side wheel.

## Steering Linkage Damping (lb-sec/in)

Linear damping of the linkage system.

## Steering Friction Lag (deg/sec)

Rotational velocity required to develop the full steering friction torque.

# Roll Steer Properties

The Roll Steer properties for the selected wheel location are displayed and edited using this dialog when the suspension type is **Solid Axle** or one of the tandem types. (For independent suspensions the full [Roll Steer Dialog Box](RollSteerDlg.md) with polynomial coefficients is displayed instead.)

The dialog title identifies the current axle (e.g., *Roll Steer: Axle No. 2 (Solid Axle)*).

This dialog contains the following item:

## Roll Steer Coefficient (deg/deg)

User-editable roll steer coefficient: the ratio of wheel (axle) steer angle to body roll angle.

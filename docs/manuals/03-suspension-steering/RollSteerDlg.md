# Roll Steer Dialog

The Roll Steer properties for the selected wheel location are displayed and edited using this dialog box. This dialog is used for **independent** suspensions; for solid and tandem axles the single-coefficient [Roll Steer Properties dialog](RollSteerNewDlg.md) is displayed instead.

The dialog title identifies the current axle and side.

## Roll Steer Graph

Graph titled *Roll Steer vs. Jounce/Rebound* showing the wheel steer angle produced by the polynomial coefficients below, plotted against suspension jounce/rebound deflection.

## Roll Steer Constant Coef (deg)

Initial steer angle at zero suspension deflection (constant term of the polynomial).

## Roll Steer Linear Coef (deg/in)

Linear coefficient of roll steer vs. jounce/rebound deflection.

## Roll Steer Quadratic Coef (deg/in²)

Quadratic coefficient of roll steer vs. jounce/rebound deflection.

## Roll Steer Cubic Coef (deg/in³)

Cubic coefficient of roll steer vs. jounce/rebound deflection.

## Apply

Redraws the graph using the currently entered coefficients.

## Print

Prints the displayed graph.

# Inertia Dialog Box

**Inertias Dialog Box**

The suspension inertial properties for the selected wheel location are displayed and edited using this dialog box. Each value may be set either with its slider or by typing directly into the edit field.

This dialog box contains the following parameters:

## Solid Axle Weight
Total weight of the axle and spring. HVE stores the corresponding mass internally; the displayed weight is calculated using the current acceleration of gravity from the Environment Editor.

## Roll / Yaw Inertia
Roll and yaw inertia of the axle. Do not include wheel inertias; they are included with the tires.

# B Stiffness Coefficient Dialog

**B Stiffness Coefficients Dialog Box**

The selected vehicle's stiffness coefficients are normally used for damage-based calculation methods. Choosing B Stiffness displays a dialog showing the current B-stiffness values for each crush zone. These coefficients are initially assigned by the Vehicle Editor when the vehicle is created and may be replaced by user-entered values for each crush zone. Each value may be set either with its slider or by typing directly into the edit field.

## Zone 1 - 9
Enter the B-stiffness coefficient for the zone. Only the zones defined in the current damage profile are enabled; the remaining zone fields are disabled.

## All Same As Zone 1
Assigns the B-stiffness coefficient of the first zone to all the zones.

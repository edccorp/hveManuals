# Inter-Vehicle Connections Dialog

The connections between tow vehicles and trailers are displayed and edited using the Inter-Vehicle Connections dialog. Program units for vehicle connection properties are shown with each field. Fields are disabled when the corresponding connection type is *None*.

## Front Connection

**Type**
The type of connection on the front of the current vehicle. The allowable types are:

- None
- Hitch
- King Pin
- Fixed Drawbar (pintle eye rigidly attached to the trailer)
- Hinged Drawbar (pintle eye on a hinged drawbar)

**Coordinates (in) - x**
The vehicle-fixed X coordinate of the front connection.

**y**
The vehicle-fixed Y coordinate of the front connection.

**z**
The vehicle-fixed Z coordinate of the front connection.

## Rear Connection

**Type**
The type of connection on the rear of the current vehicle. The allowable types are:

- None
- Ball
- Fifth Wheel
- Pintle Hook

**Coordinates (in) - x**
The vehicle-fixed X coordinate of the rear connection.

**y**
The vehicle-fixed Y coordinate of the rear connection.

**z**
The vehicle-fixed Z coordinate of the rear connection.

**Radius (in)**
The moment arm at which the frictional torque acts at the connection between the tow vehicle and trailer.

**Friction**
The friction coefficient at the connection between the tow vehicle and trailer.

**Max Articulation Angle (deg)**
The maximum articulation angle allowable between the tow vehicle and trailer.

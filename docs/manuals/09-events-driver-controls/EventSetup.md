# Event Setup Menu

The Event Set-up menu contains the following menu items. (Items appear enabled or disabled according to the capabilities of the current event's calculation method.)

**Position / Velocity Command (Ctrl+P)**
Displays the [Position/Velocity Dialog Box](PosVelDlg.md).

**Driver Controls Command (Ctrl+D)**
Displays the [Driver Controls Dialog Box](DriverControls.md).

**Damage Profiles Command (Ctrl+M)**
Displays the Damage Profiles dialog, used to display and edit the crush damage profile (damage dimensions and location) for the selected vehicle in the current event.

**Collision Pulse Command (Ctrl+U)**
Displays the Collision Pulse Dialog Box.

**Vehicle Mesh Command**
Displays the Vehicle Mesh dialog, used to display and edit the body mesh options (e.g., mesh subdivision) used by DyMESH-based collision simulations for the selected vehicle.

**Payload Command**
Displays the Payload Dialog Box.

**Wheels Command (Ctrl+W)**
Displays the Wheels Properties Dialog Box.

**Accelerometers Command**
Displays the Accelerometers Dialog Box.

**Contacts Command (Ctrl+T)**
Displays the [Contacts Dialog Box](ContactsDlg.md).

**Restraints**
Assigns restraint system usage parameters to the selected human. This popup menu contains the following items:

- **Airbags (Ctrl+G)** — Displays the Air Bag Restraints Dialog Box.
- **Belts (Ctrl+B)** — Displays the Belts Restraint System Dialog Box.

**Signals Command**
Displays the Event Signals Setup dialog, used to assign traffic signal timing for the current event.

**Notes Command**
Displays the Event Notes dialog, used to enter free-form notes describing the current event.

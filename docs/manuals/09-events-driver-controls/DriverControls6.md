# Driver Controls Filter

The Filter tab of the HVE Driver (Path Follower) page contains user-entered parameters that define driver physiological capabilities.

## Use Neuro-Muscular Filter

When checked, the driver model's steering output is passed through a neuro-muscular filter defined by the parameters below. When unchecked, the filter parameters are ignored.

## Neuro-Muscular Filter Parameters

- **Driver Lag Time (sec)** — Neuro-muscular filter lag time.
- **Driver Lead Time (sec)** — Neuro-muscular filter lead time.
- **Driver Time Delay (sec)** — Neuro-muscular filter time delay.

Each parameter is entered with a slider or by typing directly into the associated edit field.

## Notable updates relative to the 2006 printed edition

- **File menu:** "Video Set-up" has been replaced by **Video Creator**; **Export Preview Case**, **Export** (Export Variables dialog) and **FBX Export** were added.
- **Set-up menu:** "Vehicle Damage" is now titled **Damage Profiles**, "Mesh" is now **Vehicle Mesh**; **Signals** (traffic signal set-up) and **Notes** (event notes) were added; the Driver Controls dialog gained a **Lights** page.
- **Options menu:** added **Create DB**, **Show Tracks**, **Show CGs**, **Show CG Paths**, **Show Accelerometers**, **Show Accelerometer Paths**, **Show Position Sequences**, **Show Connections**, **Distance Tool**, **Show Path** and **Shadows**; **Add Playback Window** is no longer in the Options menu (playback windows are added from Playback mode).
- **Help menu:** added the **User Manuals** cascade (PDF manuals for HVE and all calculation models) and **Online Licensing** (Register User ID Code, Refresh Licenses).

# Section Seven: Playback Editor

A motor vehicle crash sequence often includes several events involving many different vehicles, and any number of occupants and pedestrians. The purpose of the Playback Editor is to combine simulations from several events into a single, coherent sequence involving multiple humans and vehicles. The Playback Editor's video interface allows the user to produce professional quality video output. In addition, the Playback Editor provides the ability to view and print the complete output reports for each event. This section of the HVE User's Manual describes how to use the Playback Editor to perform these tasks.

*(updated: the video interface now records directly to AVI movie files using a user-selected compressor, rather than to video tape; see the Video Setup notes in Chapter 17.)*

## General Procedure

To use the Playback Editor, perform the following steps:

1. Select Playback mode.
2. Create one or more Report Windows to view the results for each event.
3. If desired, print the results on the system printer.
4. Create a Playback Window to combine several trajectory simulations into a single, coherent sequence involving multiple humans and vehicles.
5. Use HVE's video interface to produce a simulation movie file.

# Wheel Properties Dialog

Clicking on a wheel of the current vehicle displays the Wheel popup dialog, from which you can navigate to the following options for the selected wheel position:

## Suspension...

Displays the Suspension Data dialog (see SuspDlg).

## Brake...

Displays the Brake Assembly dialog (see BrkAsmblyDlg).

## Tire...

Displays the [Tire Information dialog](TireInfoDlg.md).

## Wheel Location...

Displays the Wheel Location dialog (see WheelLocDlg).

## Wheel Image...

Displays a submenu for assigning a wheel (rim) image. Choose New or Open, applied to This Wheel, This Axle or All Wheels. (New since the original help.)

## Tire Sidewall Image...

Displays a submenu for assigning a tire sidewall image. Choose New or Open, applied to This Tire, This Axle or All Tires. (New since the original help.)

## Tire Tread Image...

Displays a submenu for assigning a tire tread image. Choose New or Open, applied to This Tire, This Axle or All Tires. (New since the original help.)

## Track Color...

Displays a submenu for setting the color used to draw this tire's track (tire path) marks: This Wheel, This Axle or All Wheels. (New since the original help.)

## Skid Color...

Displays a submenu for setting the color used to draw this tire's skid marks: This Wheel, This Axle or All Wheels. (New since the original help.)

## Cancel

Closes the popup without making a selection.
